# CodinGLM CLI Companion

The CodinGLM CLI Companion extension pairs with
[CodinGLM CLI](https://github.com/Dfunk55/CodinGLM). This extension is
compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: CodinGLM CLI gains awareness of the files you have
  open in your editor, providing it with a richer understanding of your
  project's structure and content.

- Selection Context: CodinGLM CLI can easily access your cursor's position and
  selected text within the editor, giving it valuable context directly from your
  current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by
  CodinGLM CLI directly within the editor.

- Launch CodinGLM CLI: Quickly start a new CodinGLM CLI session from the Command
  Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "CodinGLM CLI: Run"
  command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.99.0 or newer
- CodinGLM CLI (installed separately) running within the integrated terminal
